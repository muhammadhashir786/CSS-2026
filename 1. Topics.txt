CSS Selectors:
    1. element/type - also select multiple elements separated by "," to give same styles
    2. class "."
    3. id "#"
    4. universal "*"
    5. child ">"
    6. descendant
    7. adjacent sibling "+"    - select next sibling of your defined element
    8. general sibling "~"
    9. pseudo class selectors  - style different states of element (:hover :visited :link :active)
    10. pseudo element selectors    - select any part of an element
    11. attribute [name=value]    

    Syntax:
        1. h1, p, s {property: value;}
        2. .header {property: value;}
        3. #main {property: value;}
        4. * {property: value;}
        5. p > a {property: value;}
        6. p a {property: value;}
        7. h2 + p {property: value;}
        8. h2 ~ p {property: value;}
        9. selector: pseudo-class {property: value;}
        10. p::first-line{property: value;}
        11. a[href="url"] {property: value;}

    Examples:

        Element & Class:
            element.className, element.className {property: value;}
            
        Descendant: 
            body h1+p .special {property: value;}

        Attribute:
            li[class] {}
            li[class="a"] {}
            li[class~="a"] {}

CSS Units:
    units are used to define size of html elements according to our need

    2 Types of Units:
        1. Absolute Units   - fixed & not dependent
        2. Relative Units   - changes & dependent    

    1. Absolute Units:
            px      - pixels        (1/96 of an inch) - most used    
            mm      - millimeters 
            in      - inches        (96px & 2.54cm)
            cm      - centi-meter   (96px/2.54)
            pt      - points        (1/72 of an inch)  
            pc      - picas         (12 pt)

    2. Relative Units:
            %       - percent (dependent on parent)
            em      - multiply with parent element property
            rem     - multiply with root parent element property means <html> tag
            vw      - viewport width (device width)
            vh      - viewport height (device height)
            vmin    - minimum viewport (large screen height & small screen width)
            vmax    - maximum viewport (large screen width & small screen height)

Note: 
    minimum view-port   - mobile width & computer height
    maximum view-port   - computer width & mobile height

    vw:
        set width acc to viewport width (your device sreen width) - screen width changes but element width remains same 

    vh:
        set height acc to screen height (your device sreen height) - screen height changes but element height remains same

    em:
        parent se em ko multiply krva do
        agr parent ka font size define kia gya hai ya dafault font size ho to child element ka font size ki unit "em" parent k font size se multiply hojaiega

    rem:
        root element se "em" ko multiply krva do

CSS Box Model:
    1. Element Space:
        content space/content + padding + border space 
        margin space not included
        Show in blue color

    2. Padding Space:
        space inside element
        space btw content and border 
        p:0 = padding: 0 - Emmet
        Show in green color

    3. Border Space:
        space b/w padding & margin
        Show in yellow color

    4. Margin Space:
        space outside border
        m:0 = margin: 0 - Emmet
        Show in orange color

    CSS Box Model Properties:
        box-sizing:
            Values:
                1. border-box  
                2. content-box  

            1. Border Box: 
                    a. add padding and border inside the boundry of content space,
                        means total height and width does not increase after adding padding and border 
                    b. it sets horizontal bar problem in webpage  
                    c. Exclude margin space from total Height/Width space of an Element -----------

            2. Content Box:
                    a. add padding and border outside the boundry of content space,
                        means total height and width increases after adding padding and border 
                    b. Exclude margin space from total Height/Width space of an Element -----------

                Without "border box" total height/width space includes:
                    content + padding + border + margin 

                Note:
                    margin dono m border k bahir se add hoga

    Margin Collapsing:
        When margin of 2 elements will collapse, only max margin will apply

CSS Background properties:
    1. Background Color
            background-color: colorValue;

    2. Background Image:
            background-image: url("");

    3. Background Repeat:
            background-repeat: repeat|repeat-x|repeat-y|no-repeat|initial|inherit|space|round;

            "repeat" is default

    4. Background Position:
            background-position: value; 

            Values:
                if specify one value other will be "center"
                    left top
                    left center
                    left bottom
                    right top
                    right center
                    right bottom
                    center top
                    center center
                    center bottom

                    x y         - if specify one value other will be 50%
                    x% y%       - if specify one value other will be 50%
                    initial
                    inherit

                "0%" is default

    5. Background Size:
            background-size: auto|length|cover|contain|initial|inherit;

            Note:
                value "length" is height and width - if one specify other will set to "auto"
                auto is default

    6. Background Attachment:
            background-attachment: scroll|fixed|local|initial|inherit;

            "scroll" is default
